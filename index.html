Preliminary Design of a Rock Paper Scissors Game interface

The assignment was to design ( not implement ) an interface to play the stone-paper-scissors game. The prototype was designed for a tablet using Balsamiq Mockup tools.

The following is the help screen which can be accessed via the menu option in the lower left corner of the screen.

Once the players select the option, the chosen options get highlighted on their respective screens. Player 1 cannot see the option chosen by Player 2 on his screen and vice versa.
